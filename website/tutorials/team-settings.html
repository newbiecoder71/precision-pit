How To Set Up Team Settings

This guide helps a team owner finish the important settings after the team is created. It is written as an editable draft first, so you can revise the steps before making it a final PDF.

Open The Settings Area

  1. Sign in to Precision Pit with the team owner account.
  2. Open the Settings section from the app navigation.
  3. Review the team details that will affect the race-night workflow.

Complete The Team Profile

These details help the app show the right experience for the team.

  • Confirm the team name and owner details are correct.
  • Select the racing type and class information the team uses.
  • Review any baseline configuration details that should be saved early.
  • Make sure the settings match the car and race program the team will actually run.

Save Baseline Information

New teams should save useful baseline information before their first race night.

  1. Open the baseline setup area.
  2. Enter the saved chassis, suspension, tire, and driveline information the team uses most often.
  3. Save the details so new race nights can start from a dependable baseline.
  4. Review the stored information once to make sure it looks right before leaving the screen.

Best Practice

Teams usually get the most value from Precision Pit when they finish settings before inviting everyone else in. That way the crew joins an account that already reflects the team name, racing details, and saved setup structure.
